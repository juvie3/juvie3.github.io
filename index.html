Technologies

I am just starting out, but so far these are the technologies I now have in my tool belt. But I'm constantly learning more and more.

React

I use React as my primary frontend stack, as I am very familiar with utilizing components to keep my code DRY and I consider creating web forms as one of my specialties.

Redux

I like to take advantage of the many benefits of having a Redux store instead of just using React context. I know how to create and maintain a store, as well as normalize the data.

Flask / Express

Flask is my preferred backend stack when writing in Python, but I am very familiar with Express and Node.js if JavaScript is the language required for the backend server.

WebSocket

Sockets are such a fascination for me because it's bi-directional and it's just always open. I can deploy sockets in many different ways, depending on the web application.

AWS

If Amazon Web Services is needed for a website, then I am familiar with handling the implementation and maintaining the use of their database storage system.

Databases

Creating and maintaining a database is actually fun for me. Raw SQL is fascinatingly complex, but I am familiar with SQLite3, Sequelize, SQLAlchemy, and PostgreSQL.
